Description official descriptions

True to the aircraft‚Äôs flexibility, you pilot the F-16 in assignments as diverse as dogfights, bombing raids or reconnaissance missions. It is up to you to plan your flight route by setting waypoints on the tactical map prior to take-off; you can also (but need not) select the armament of your plane. When you have accumulated enough flight experience, you may command entire squadrons in the fictional Operation Conquest campaign, a simple simulation of an entire war theater that already hinted at what would later become DI‚Äôs Virtual Battlefield.

F-16 Combat Pilot allows multiplayer duels via modem or serial link.

Trivia

Media

The DOS release of the game contains 3 disks, but installation requires only two of the three.

Use Disk 1 plus either Disk 2 or 3, depending upon which graphics you need. Disk 2 is for EGA and Tandy graphics, while Disk 3 is for CGA & Monochrome.
