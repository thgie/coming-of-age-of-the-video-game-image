Description

Raster Blaster is one of the very first computer pinball simulations, created by Bill Budge. It was the foundation for Budge's later hit Pinball Construction Set and is an unofficial adaptation of the Williams pinball table Firepower.

The game has two difficulties - in Easy mode a ball shield is always present, but it must first be activated in Hard mode - and up to four players can play the game, taking turns.
