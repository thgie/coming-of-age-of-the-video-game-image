Description official description

Soccer replaced the game Football in European export versions of the Magnavox Odyssey. It uses a custom screen overlay and two game cards, #3 and #5.

The gameplay is a variant of Table Tennis in which the horizontal position of players is fixed (although not enforced by the Odyssey console itself). If a player manages to maneuver the ball around her opponent, the next volley will take place one step closer to the opponent's goal. Once a player has reached a position in the opponent's half of the playing field, she is allowed to score a goal. The game also includes rules for penalty kicks, which require changing the game card.
