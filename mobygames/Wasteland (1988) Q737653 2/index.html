Description official description

Wasteland: Remastered is a remake of the 1988 role-playing game Wasteland. It features entirely new graphics and audio, expanded soundtrack, the physical storybook entirely brought in-game with illustrations and voiced narration, modern controls including gamepad support, and other quality of life improvements while retaining the core game design as it was in the original game.
