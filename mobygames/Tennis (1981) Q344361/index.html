Description official description

Tennis is a singles tennis game which can be played by one player against the computer or two players against each other. The game follows standard tennis rules, and allows players to hit a variety of forehand and backhand shots to any location on the court. There are two skill levels included; a slower practice game, and the more challenging full speed game.

Trivia

Graphics

When you play NTSC cartridges on a PAL TVs that supports the mode, some colours can appear wrong. In this case the court appears blue rather than green - almost as if the game has automatically modernized from grass to hard-court surface (obviously the ball still behaves as if skimming off grass though).

Activision patch

Players who won one set against the computer could send in a photo of their TV screen and get an Activision Tennis patch.
