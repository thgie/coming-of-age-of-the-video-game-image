Description official description

Jail Break is a game based on the Arcade game Breakout where you control a bat at the bottom of the screen and you deflect a ball upwards to hit and remove bricks. Once all bricks have been removed you then move onto the next level. If the ball passes the bat and hits the bottom of the screen you lose one of five lives. You can have a practise before playing the main game which allows you to have an unlimited supply of lives. The keyboard or a joystick is used to control your bat.
