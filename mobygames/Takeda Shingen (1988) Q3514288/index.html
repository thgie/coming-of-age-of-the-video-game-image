Description official description

Takeda Shingen is a 2D beat'em up game with weapons. Game is similar to its namesake on the Turbografx-16, but has some differences.

The "Sengoku" period was a time of great internal conflicts in Japan. Takeda Shingen was a famous daimyo (territorial lord) in this time - he is the game's main hero. The game is a typical beat 'em up with weapons and a scrolling screen. Controls are similar to Double Dragon II: The Revenge - with right and left attack buttons.

The title also has a between-level "minigame" - Takeda rides on his horse and must shoot a target, using his bow.
