Description

This game lets players play the Japanese board game Shōgi. Shōgi is a game that is played in a similar way as chess. Players take turns moving their pieces in a strategic manner with the main goal of capturing their opponents pieces and putting their opponents king piece into checkmate.

This game features several different game modes. These different game modes offer different opponent difficultly levels, timed games, and also some which let the computer player make the first move instead of the player.
