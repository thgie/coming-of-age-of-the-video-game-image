Description

3-D Docking is a game where the player has to steer a space ship through an asteroid field and reach a docking station. The screen is split in two and shows both a top view and a side view of the asteroid field. In the side view the player can move the ship up, down, forward and reverse while in the top view movement can only be done left and right. The game uses thruster controls where one button press causes the ship to move until the opposite key is pressed. A collision occurs when hitting an asteroid in both the top view and the side view and the game then has to be restarted. It is also possible to run out of fuel which happens when using too much thrust. When reaching the docking station a score is given based on the time it has taken.
