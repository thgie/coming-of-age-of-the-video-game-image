Description

Confrontation is a two-player wargame between blue and red forces.

The game is slown on a static map. The player controls the following units in the game: tanks, mechanized infantry, engineer, flak, anti-tanks, scout, infantry, artillery, paratroops, fighter plane, ground attack plane, transport plane. Each of them have different abilities and can perform unique tasks. Fuel is exhausted in planes, and air units require airfield on the map to start from. Terrain effects are also taken into account for unit's action.

There is one scenario provided with master program, which is set in a modern European theatre of war with two countries seeking to control the lowland and cities between the rivers. The scenario provides for both land and air forces, together with paratroopers. The suggested objective is that the winner should be the first player to control cities.

It is possible to change the forces available to each side. However, neither side may have more than 35 units. An in-game editor allows maps to be created and edited.
