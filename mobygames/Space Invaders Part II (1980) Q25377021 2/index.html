Description

Space Invaders: Part II (aka Deluxe Space Invaders) is the first sequel to Space Invaders. The basic gameplay is the same as in the original. The player controls his laser base at the bottom of the screen and has to kill the alien invaders that move back and forth above him. Some new features to the game has been added though. There is a new type of UFO that blinks and can only be shot when it is solid and UFOs can now drop new aliens onto the screen. There is also a new kind of alien that splits into two when shot. New to the presentation are short animated intersections that appear between waves.

Some minor differences exist between the original Taito release (Space Invaders: Part II) and the later Midway release (Deluxe Space Invaders). The new blinking UFO is worth 200 points instead of 500 and the splitting aliens start appearing in wave two rather than wave four. Use of colour is also different. Taito's version uses a colour monitor while Deluxe is black and white with a colour overlay.
