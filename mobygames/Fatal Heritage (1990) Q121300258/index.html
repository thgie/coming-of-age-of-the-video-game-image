Description

Fatal Heritage lives up to it's name: the protagonist of the game inherits a farm including a big treasure. Unfortunately said farm is far away and the local crime syndicate has an interest in the treasure, too. The game takes place in various locations all over the world.

The game is played in first-person with static screens which want to be explored with the mouse cursor. The interaction with the game world uses a traditional graphic adventure approach: first the action is chosen on the icon bar and then the player clicks on an object in the game world. This way the player solves the mostly inventory based puzzles, e.g. finding the misplaced will. There are also some multiple-choice conversations and friends which may accompany the player for a limited time. Those have special abilities, insights and objects which are important to solve some puzzles. The inventory is limited to 15 objects and when the player wants to take an additional one, an old has to be permanently destroyed which opens up the way for dead ends.

There is a constant time limit: a bar at the top of the screen shows how near the syndicate killer are. If the time is up, the player needs to finish the current location or the killers will end the game violently. To earn money, there are a few cards and dice mini games to participate in.
