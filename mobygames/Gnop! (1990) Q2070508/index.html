Description

Gnop! is a variant of Pong that is most notable for being the first game developed by Bungie Software. The gameplay stays close to the original formula. The player controls a paddle on the right side of the screen and has to move it up and down to direct a ball past the AI controlled paddle at the other side of the screen. The rules of Gnop! differ somewhat from traditional Pong in that the score system is similar to Volleyball. The player therefore can only score when he is serving. The game allows the player to modify some parts of the game. Both the size of the paddles and the size and speed of the ball can be changed. It's also possible to change the ending score and which player serves first. There are three different difficulty levels.

Trivia

Source Code Release

Although the game itself was released for free, the source code for the game could be ordered for $15. The game was written in Think C and according to the author included good examples of Macintosh toolbox techniques such as menus, dialogs, graphics, animation, sound and CopyBits.
