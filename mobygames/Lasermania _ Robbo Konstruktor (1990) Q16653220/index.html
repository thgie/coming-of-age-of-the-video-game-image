Description

This compilation includes the following titles: * Lasermania is a puzzle game. The aim is to destroy all the alarm system sensors by means of a laser beam and collect sets of storage. The direction of the laser beam can be changed by moving polished steel elements. Players must be wary of the laser switches which detonate ray generator. After clearing the room must accompany the vehicle to lock. The game includes over 50 levels.

  • Robbo Konstruktor is a planets editor to the game Robbo.
