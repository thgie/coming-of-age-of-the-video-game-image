Description official descriptions

An air traffic control simulator.

Guide planes across the screen through the USA, from varying airports. Graphics resemble a radar view. You must set the flight level (height), speed and direction of the aircraft passing through your jurisdiction and guide them to their correct airports. Do try and keep them apart!

They don't make 'm likes this anymore.

The Good
Surprisingly simple to learn but hard to master. Like tetris and pacman.

This game is one of the first Real Time Strategy games ever. Maybe even the first. And it's a simulation as well.

First truely commercial game to use computer generated voices.

The Bad
Nothing.

The Bottom Line
If you're interested in Air Traffic Control, or in simple but addictive games that get complicated pretty fast then this is for you.

Air Traffic Control Sim

The Good
Its quick to learn, and if you have at least a very basic interest in aviation, then it quickly becomes addictive.

The Bad
The realism could have done with a bit more thinking, If you were really controlling these aircraft then I think you could run them much closer together. The screen needs to be much bigger. But Hey, this game is nearly 20 years old...... so with that in mind......... there is nothing i don't like about this game

The Bottom Line
You have to think fast, plan ahead, I think of it as Airplane Chess, I would love to find a modern version of this game for the PC.
