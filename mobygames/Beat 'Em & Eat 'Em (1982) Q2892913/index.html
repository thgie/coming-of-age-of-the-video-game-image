Description

This adult game is reminiscent of Activision's Kaboom!.

As a man at the top of the building masturbates and ejaculates, you, as the player, must catch all the drops in your mouth(s) (see Game variations). You start with four lives and lose one if you miss a drop, which will then hit the ground.

Game variations

Game 1 has two women to catch with

Game 2 has only one woman to catch with.

Difficulty switches

Only the left switch is used.

Set to A and a catch counts if you touch a drop as far down as your waist. Set to B and a catch only counts as far down as shoulder height.

Scoring

You get one point for each drop caught.

You get an extra life every 69 points.
