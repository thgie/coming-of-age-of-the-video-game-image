Description official description

Boxing is an action boxing game for one or two players which is played from an overhead point of view. The game lasts for two minutes, and the player who can earn the most points in that time wins. You earn one point for a long jab, two points for a close power punch, and if you can manage knockout it's worth one hundred points. Two difficulty levels are included which can be set individually for each player.

This game rocks!

The Good
This is game is so fun, I almost don't want to give up my Atari. My brother and I spent (and still spend) late nights trying to knock each other out. Even though this game is older than I am, it is still a blast to play.

The Bad
Well, there's not a whole lot of criticism you can write about an Atari game as old as this one. Because the whole Atari is so old, you can't say something like, the gameplay was terrible, because all you have to work with is a joystick and a button. So, in context to 1980, I will say that this game has almost perfect gameplay. Sound was almost non-existent, but who really needs soundtrack anyway?

The Bottom Line
It's too bad no one has an Atari these days. Otherwise, I would recommend you go out and buy this game. It is, in 80's lingo, rad! It still has it's touch, even after 24 years.

Atari 2600 · by Mullet of Death (592) · 2004

Trivia

Cancelled successor

A arcade boxing game named Boxing was developed, tested and cancelled in 1978. It was developed by Mike Albaugh and it is unknown what aspects were inherited by Boxing. The perspective of Boxing was rotated by 90 degrees.
