Description official description

A top-down view, vertically scrolling, full color Formula One-style racing game. Few games before this had colored scrolling backgrounds (Taito's Super Speed Race is one). The game has notable sound effects such as a fanfare to start the race, engines, crashes, etc. The track has a narrowing/widening visual effect and a night driving mode where the headlights of cars are on.

The basic version of the game was displayed using a normal upright arcade cabinet, while the cockpit deluxe version featured a full cockpit with racing chair. The later Pro version featured a different cabinet design and added points for overtaking other cars on the screen.
