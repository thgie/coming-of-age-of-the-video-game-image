Description

Pharaoh's Revenge is a fast action computer game in the arcade tradition. Using the keyboard or joystick the player must attempt to evade an opponent (computer or human) while solving a complex maze that, when solved, leads to safety.

There are 200 levels in the main game, with 50 advanced levels that become available after the player completes the original 200. In addition, there is a construction set for creating additional levels which can then be shared with friends.
