Description

Mission: Escape! is a Lunar Rescue clone and has ten levels. The player has to land a space ship on platforms and then escape through an asteroid field to the command ship. In addition to thrust (accelerate), the player has a limited number of missiles for defense. Platforms often need to be revisited multiple times to stack points. While going up and down, enemies pass by in a horizontal path.
