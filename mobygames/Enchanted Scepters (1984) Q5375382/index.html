Description

This is one of the few adventures created specifically for the Mac. The evil Hurks are amassing an army and only one ray of hope exists - Elron the wizard can repel the attack, but only if he possesses the four enchanted scepters, lost decades ago in the Great Wars. This, of course, is where you come in. Find the scepters and return alive!

The game has over 200 scenes with interactive graphics - doors open, objects move, and so on. It also features what the publisher calls 'RealSound' - basically real digitized sound effects and not synthesized. While this can be played like a standard text adventure, this also features one of the earliest point and click interfaces in adventure games. In this case a series of drop-down menus you can select commands from.
