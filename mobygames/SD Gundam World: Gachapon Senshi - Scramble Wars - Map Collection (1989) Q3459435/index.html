Description

SD Gundam World: Gachapon Senshi - Scramble Wars Map Collection is a special version of the original SD Gundam World: Gachapon Senshi - Scramble Wars replacing the original scenarios with ten new scenarios. It could only be obtained via Famicom Disk Writer kiosks. Gameplay and graphics remain unchanged.
