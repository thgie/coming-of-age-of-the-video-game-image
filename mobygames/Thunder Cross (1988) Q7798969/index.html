Description official descriptions

Thunder Cross is a 2D side-scrolling shoot 'em up game where you take control of the spaceship Blue Thunder M-45 in a space themed environment. You have to stop an evil organization called Black Impulse from conquering the galaxy.

As you make progress and fight of various space crafts, static guns, walkers and bosses power-ups come available. Power-ups include increased speed and/or bullets. A boomerang laser, twin laser and ups. Special power-ups can only be used with an option power-up capsule and include a macro laser, nuke bomb and flamethrower. The game can be played solo or with a friend witch then takes control of the spaceship Red Thunder M-24.
