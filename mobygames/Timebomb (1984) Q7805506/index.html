Description

Timebomb is a simple arcade game (but thinking is very helpful). The player must diffuse six timebombs, before the time runs out. He can only move on blue squares (disappearing after single stepping - player can stuck with no any possibility of moves), and must avoid the skulls and boots. Each bomb a has time limit. The player has 3 lives - if he lost all lives, the game is over.
