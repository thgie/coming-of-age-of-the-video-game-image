Description official description

Galactic Attack is a real-time space combat game which combines arcade elements with strategic planning. Using keyboard commands you control speed and direction as you seek to recapture the planets of the solar system from invaders. The primary weapons (torpedoes) and the secondary weapons (phasers) are fired in the direction entered at the keyboard.
