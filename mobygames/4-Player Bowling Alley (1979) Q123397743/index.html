Description

4 Player Bowling Alley is a bowling game for up to four players. The bowling ball is controlled with a track ball and the bowling lane is shown on the screen with all the pins using a top-down view. The game features two game modes: "Regulation" is a normal game of bowling and "Flash" features a timing element with a moving target.
