Description

Eggerland: Sōzō e no Tabidachi (English: Departure to Creation) is a Famicom Disk System game in the Eggerland/Adventures of Lolo series. It contains 50 stages, many of which are copies from previous Eggerland games.

You play as Lolo, a small, ball-like creature that must solve screen-based puzzles. Each level contains hearts that Lolo must collect to open a chest and collect the key. Lolo must strategically move blocks and manipulate enemies to accomplish this feat. Many enemies wait until after the last heart is collected to spring to life, splitting the challenge between collecting all the hearts and then still getting the key.

New to this game is a map creation mode (hence the title, Departure to Creation) that allows players to design their own puzzles. The player have access to the same enemies and obstacles in the normal game, and can save their puzzles to disk.
