Description official description

SoftAid is a charity compilation of ten games with all proceeds going to The Bob Geldof Band-Aid Ethiopian Appeal Fund:

  • Gumshoe
  • Pitfall
  • Startrader
  • Kokotoni Wilf
  • China Miner
  • Gilligan's Gold
  • Fred
  • Gyropod
  • Falcon Patrol
  • Flak

Included in the compilation was a copy of the Single 'Do They Know it's Christmas' by Band-Aid
